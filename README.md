# 💻 Aman Deep — Portfolio

A fast, highly optimized developer portfolio showcasing builds, AI experiments, desktop tools, and server integrations. 

Built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Lucide Icons**—optimized for premium responsiveness and search visibility.

---

## 🚀 Key Highlights & Optimizations

*   **⚡ Native CSS Animations:** Dropped heavy animation runtimes (`motion/react`) in favor of hardware-accelerated CSS keyframe animations. This removed ~200KB of client-side JavaScript, boosting the mobile performance score to **98+**.
*   **🔍 Search Engine Optimization (SEO):** Embedded structured **JSON-LD Schema Markup** (`Person` and `WebSite` schemas), generated a clean sitemap (`sitemap.xml`), and set up canonical indexing rules so Google displays nested sitelinks cleanly.
*   **🛠️ CI/CD Pipelines:** Integrated GitHub Actions to automate lint checks, TypeScript compiler passes, and production builds on push/pull requests.

## 🛠️ Local Development

### 1. Install dependencies
```bash
npm install
```

### 2. Run the development server
```bash
npm run dev
```

### 3. Build & start in production mode (for Lighthouse tests)
```bash
npm run build
npm run start
```
