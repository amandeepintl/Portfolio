## Features

- **Dark Mode** — Sleek dark theme with a custom teal color palette
- **Glassmorphism Cards** — Frosted-glass UI cards with blur and subtle borders
- **Apple-Inspired Corners** — Smooth, large border-radius throughout the design
- **Custom Cursor Dot** — A smooth teal dot that trails the mouse and scales on hover
- **Typewriter Effect** — Rotating tagline in the hero section
- **Scroll Animations** — Elements reveal on scroll with staggered timing
- **Animated Progress Bars** — Skill proficiency bars animate into view
- **Responsive** — Fully responsive across desktop, tablet, and mobile
- **SVG Icons** — Clean inline SVG icons, no external icon libraries
- **Zero Dependencies** — Pure HTML, CSS, and JS with no build step

## Getting Started

No install needed. Just clone and open:

```bash
git clone https://github.com/amandeepintl/Portfolio.git
cd Portfolio
```

Then open `index.html` in your browser.

You can also deploy directly to **Netlify**, **Vercel**, or **GitHub Pages** since it is all static files.
