# Aman Deep Portfolio

Vercel-ready portfolio built with Next.js, TypeScript, Tailwind CSS, Motion, and Lucide icons.

## Edit content

Most portfolio text and project data lives here:

```txt
src/data/portfolio.ts
```

Update that file when you want to change your intro, links, projects, learning tracks, or case notes.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push this folder to GitHub and import the repo in Vercel. The default Vercel settings should work.
